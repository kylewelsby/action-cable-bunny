## Installation

Add this line to your application's Gemfile:

```ruby
gem 'action-cable-bunny'
```

And then execute:

```bash
bundle
```

## Usage

```yaml
# config/cable.yml
bunny: &bunny
  adapter: bunny
production: *bunny
development: *bunny

```

## Development

After checking out the repo, run `bundle install` to install dependencies. Then, run `bundle exec rake` to run the tests.
